# 📱 Jarvis - Mobil Hızlı Başlangıç Rehberi

Jarvis'i iPhone veya Android telefonunuzda kullanmak için sadece **3 adım** gereklidir!

---

## 🍎 iPhone Kullanıcıları

### Adım 1: Streamlit Cloud'a Yayınla (Bilgisayarda)

```bash
# 1. GitHub'da yeni repository oluşturun
# 2. Proje dosyalarını yükleyin
# 3. https://streamlit.io/cloud adresine gidin
# 4. GitHub hesabınızla giriş yapın
# 5. "New app" → Repository seçin → "Deploy"
```

### Adım 2: Linkini Kopyala

Deployment tamamlandıktan sonra, Streamlit Cloud'un verdiği linki kopyalayın.
(Örn: `https://jarvis-ai.streamlit.app`)

### Adım 3: Ana Ekrana Ekle

1. **Safari'de** linki açın
2. Alttaki **Paylaş** simgesine (⬆️) dokunun
3. **Ana Ekrana Ekle** seçeneğini seçin
4. Adı girin ve **Ekle** butonuna basın

✅ **Bitti!** Artık ana ekranınızda Jarvis ikonu var!

---

## 🤖 Android Kullanıcıları

### Adım 1-2: iPhone ile Aynı

(Streamlit Cloud'a yayınlayın ve linki kopyalayın)

### Adım 3: Ana Ekrana Ekle

1. **Chrome'da** linki açın
2. Sağ üstteki **⋮** (üç nokta) menüsünü açın
3. **Ana ekrana yükle** seçeneğini tıklayın
4. Adı girin ve **Yükle** butonuna basın

✅ **Bitti!** Artık ana ekranınızda Jarvis ikonu var!

## 🚀 Uygulamayı Kullan

### Temel Komutlar

```
"Merhaba Jarvis!" → Sohbet başlat
"Saat kaç?" → Mevcut saati göster
"Bugünün tarihi nedir?" → Tarihi göster
"Sistem promptunu değiştir: Senin adın Doktor" → Kişiliğini değiştir
```

### Sidebar Ayarları

- **API Anahtarı:** İlk kez giriş yapıyorsanız OpenRouter API anahtarınızı girin
- **Sistem Promptu:** Jarvis'in davranışını özelleştir
- **Sohbeti Temizle:** Sohbet geçmişini sil

## 🔑 OpenRouter API Anahtarı Alma (1 dakika)

1. https://openrouter.ai adresine gidin
2. "Sign Up" → E-mail ile kayıt olun
3. Dashboard → "Keys" → "Create New Key"
4. API anahtarını kopyalayın
5. Jarvis'in sidebar'ına yapıştırın
